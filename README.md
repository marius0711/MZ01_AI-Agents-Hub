# Research Agent

An autonomous agent that reads articles or PDFs, summarizes them, and creates flashcards.

## Features
- Automatic loading of URLs or PDFs  
- Chunking and summarization with GPT/Claude  
- Generation of flashcards in JSON and Markdown formats  

## Setup
```bash
pip install -r requirements.txt
export OPENAI_API_KEY="your-key"
python main.py
```
